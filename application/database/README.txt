
Steps to initialize the DB in you local: 
1. installation of docker, refer: https://docs.docker.com/engine/install/
2. open your terminal and go to database/ folder
3. run command 'docker compose up'
4. you shoud be able to see last message line like "database-postgres-1  | 2023-06-22 03:03:56.063 UTC [1] LOG:  database system is ready to accept connections"
